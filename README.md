# ⛄ アメダスマップ (AMeDAS_Visualizer)

気象庁の公開データを利用して、日本全国のアメダス観測所における現在のデータを地図上に可視化するWebアプリケーションです。 現在は積雪深のみ対応しています。HTMLファイル1つで動作する軽量なシングルページアプリケーション（SPA）として設計されています。

## ✨ 主な機能

### 地図で見る積雪状況

全国の積雪量が地図上に丸いマークで表示されます。

雪が多いほどマークが大きく、色が濃くなるため、どこで大雪になっているかが一目でわかります。

雪が積もっていない（0cm）観測所もグレーで表示されるので、雪の有無も確認できます。

### 詳しい情報の確認

地図上のマークを押すと、観測所の名前と詳しい積雪量（cm）が表示されます。

「推移グラフ」ボタンから、気象庁の公式サイトで過去の雪の降り方を確認できます。

### お気に入り登録

よく見る場所を「お気に入り」に登録できます。

お気に入りに登録した場所は、地図上で常に積雪量の数字が表示されるようになり、チェックしやすくなります。

### ランキングとリスト

* **全ての地点**: 全国の積雪量が多い上位20地点をランキングで見ることができます。

* **お気に入り**: 登録した地点の積雪量を一覧リストで見ることができます。

リストを押すと、その場所に地図が自動で移動します。

### スマホ・PC対応

パソコンだけでなく、スマートフォンやタブレットでも見やすく表示されます。

画面左上のパネルは折りたたむことができるので、スマホの画面でも地図を広く見ることができます。

## 🛠️ 技術スタック

* **HTML5 / JavaScript（ES6+）**

* **Leaflet.js**: 地図描画ライブラリ

* **Tailwind CSS**: スタイリング（CDN版）

* **Data Source**: 気象庁（JMA）非公式API / GeoJSON

## 🚀 使い方

### 1. インストール

リポジトリをクローンするか、`index.html` をダウンロードしてください。

### 2. 実行

`index.html` をブラウザ（Chrome, Firefox, Safari, Edgeなど）で開くだけで動作します。

特別なWebサーバーやビルドプロセスは不要です。

※インターネット接続が必要です。

## ⚠️ 注意事項

### データ出典について

本アプリケーションは、気象庁のWebサイトで公開されているデータを利用しています。

> 出典: 気象庁 Japan Meteorological Agency

本アプリは気象庁の公認アプリではありません。

### CORSプロキシについて

気象庁のデータサーバーへブラウザから直接アクセスする際のCORS（Cross-Origin Resource Sharing）制限を回避するため、デフォルトで [corsproxy.io](https://corsproxy.io/) を経由する設定になっています。

本番環境や高負荷な環境で運用する場合は、ご自身でプロキシサーバーを用意するか、適切なサーバー設定を行ってください。

コード内の const PROXY_URL = 'https://corsproxy.io/?'; を変更することで設定可能です。

### 🗺️ データ更新について

アメダスのデータは通常10分ごとに更新されます。

自動更新ではありません。最新情報を確認したい場合は、画面上の「データを更新」ボタンを押してください。
